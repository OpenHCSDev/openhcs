\subsection*{Mechanization Contract}

To match the paper-4 standard, each theorem-level handle is typed as one of:
\begin{itemize}
\item \textbf{Full:} direct Lean theorem support.
\item \textbf{Full (conditional classical):} Lean closure theorem parameterized by explicit information-theory assumptions.
\item \textbf{Full (derived/model-level):} theorem-level closure by composition from mechanized cores in the stated model.
\end{itemize}
