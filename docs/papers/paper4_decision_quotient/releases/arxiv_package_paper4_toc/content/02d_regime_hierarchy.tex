\section{Regime Hierarchy}\label{sec:regime-hierarchy}

We establish the strict inclusion hierarchy among decision regimes.

\begin{center}
\begin{tabularx}{\linewidth}{@{}ccl@{}}
\toprule
\textbf{Inclusion} & \textbf{Complexity Classes} & \textbf{Condition} \\
\midrule
Static $\subset$ Stochastic & \coNP $\subset$ \PP & Standard assumptions (P $\neq$ \coNP) \\
Stochastic $\subset$ Sequential & \PP $\subset$ \PSPACE & Standard assumptions (P $\neq$ \PP) \\
\bottomrule
\end{tabularx}
\end{center}

\begin{claim}{Strict inclusion static $\subset$ stochastic}{prop:static-stochastic-strict}
Under standard complexity assumptions (P $\neq$ \coNP), there exist stochastic decision problems that are not statically reducible.
\claimstamp{P\ref{prop:static-stochastic-strict}}{P $\neq$ \coNP}
\end{claim}
\leanmeta{\LH{DC1}.}

\begin{claim}{Strict inclusion stochastic $\subset$ sequential}{prop:stochastic-sequential-strict}
Under standard complexity assumptions (P $\neq$ \PP), there exist sequential decision problems that are not stochastically reducible.
\claimstamp{P\ref{prop:stochastic-sequential-strict}}{P $\neq$ \PP}
\end{claim}
\leanmeta{\LH{DC2}.}

\subsection{Integrity-Resource Bound per Regime}

The integrity-resource bound from Paper 4 generalizes to each regime:

\begin{enumerate}
\item \textbf{Static} (\coNP-hard): polynomial-time agent must abstain on some instances
\item \textbf{Stochastic} (\PP-hard): polynomial-time agent must abstain on MORE instances
\item \textbf{Sequential} (\PSPACE-hard): polynomial-time agent must abstain on EVEN MORE instances
\end{enumerate}

Formally, if $C_1 \subsetneq C_2$ (complexity classes), then the set of instances where integrity forces abstention under $C_1$-resources is a strict superset of those under $C_2$-resources:

\[
\{\sigma : \text{integrity forces abstention at } C_1\}
\supsetneq
\{\sigma : \text{integrity forces abstention at } C_2\}
\]

\begin{claim}{Abstention frontier shifts}{prop:abstention-frontier}
As regime complexity increases (static $\to$ stochastic $\to$ sequential), the abstention frontier expands.
\claimstamp{P\ref{prop:abstention-frontier}}{RG}
\end{claim}

\leanmeta{\LH{CC39}}

\subsection{Regime Detection}

A meta-problem: given a problem instance, which regime is it in?

\begin{decision}\label{dec:regime-detection}
\textbf{REGIME-DETECTION}: Given a problem instance, is it static, stochastic, or sequential?
\end{decision}

Conjecture: regime detection is polynomial for the represented family:
\begin{itemize}
\item Bounded actions $\to$ static
\item Distribution present $\to$ stochastic  
\item Transitions/observations present $\to$ sequential
\end{itemize}
