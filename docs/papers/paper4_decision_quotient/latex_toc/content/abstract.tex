We study physical information sufficiency: given a decision problem $D=(A,S,U)$ with factored state space $S=X_1\times\cdots\times X_n$, does coordinate set $I$ determine optimal action? Formally $s_I=s'_I\implies\Opt(s)=\Opt(s')$.

Complexity: SUFFICIENCY-CHECK is coNP-complete; MINIMUM-SUFFICIENT-SET is coNP-complete; ANCHOR-SUFFICIENCY is $\Sigma_2^P$-complete. Six polynomial-time regimes under explicit-state. Succinct with ETH gives $2^{\Omega(n)}$.

Physical: formalizes encoding $E:\mathcal{P}\to\mathcal{D}$ with transport rules. Atoms with orbital transitions are agents.

Keywords: computational complexity, physical information, decision theory, polynomial hierarchy, formal verification
