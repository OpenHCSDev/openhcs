We study \emph{physical information sufficiency} as a decision-theoretic meta-problem. For \(\mathcal{D}=(A,S,U)\) with factored state space \(S=X_1\times\cdots\times X_n\), a coordinate set \(I\) is sufficient when
\[
s_I=s'_I \implies \Opt(s)=\Opt(s').
\]
This asks whether projected information preserves the full optimal-action correspondence.

\textbf{Complexity landscape.}
\begin{itemize}
\item SUFFICIENCY-CHECK is \coNP-complete; MINIMUM-SUFFICIENT-SET is \coNP-complete; ANCHOR-SUFFICIENCY is \(\SigmaP{2}\)-complete. Informally, this gap separates verification (checking whether a given coordinate set suffices) from discovery (finding a grounding under which the optimal action is determined), and suggests the latter is structurally harder unless the polynomial hierarchy collapses.
\item Under explicit-state encoding, polynomial-time algorithms hold for bounded actions, separable utility, and tree-structured utility.
\item Under succinct encoding, hardness is regime-dependent: with ETH, there are worst-case families with \(k^*=n\) requiring \(2^{\Omega(n)}\) time.
\item Under query access, the finite-state core has worst-case Opt-oracle complexity \(\Omega(|S|)\), with Boolean value-entry and state-batch refinements preserving the obstruction.
\end{itemize}
