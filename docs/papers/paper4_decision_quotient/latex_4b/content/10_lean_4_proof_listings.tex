\subsection{What Is Machine-Checked}

The Lean formalization establishes:

\begin{enumerate}
\item \textbf{STOCHASTIC-SUFFICIENCY PP-completeness}: Reduction from MAJSAT showing stochastic sufficiency checking is PP-complete.

\item \textbf{SEQUENTIAL-SUFFICIENCY PSPACE-completeness}: Reduction from TQBF showing sequential sufficiency checking is PSPACE-complete.

\item \textbf{Substrate independence theorems}: Formal proofs that the matrix verdict is $\kappa$-independent while trajectories are $\kappa$-dependent.

\item \textbf{Bridge transfer conditions}: Formalized conditions under which sufficiency transfers between regimes.
\end{enumerate}
